------------------------------------------
PROJECT DESCRIPTION
------------------------------------------
This is a modern and responsive Library Management System
that allows librarians to manage books, issue records, and
returns efficiently. The project uses only front-end
technologies and stores data locally using LocalStorage.

------------------------------------------
FILES INCLUDED
------------------------------------------
index.html        - Splash screen
login.html        - Librarian login with CAPTCHA
dashboard.html    - Main dashboard (books, issue, return)
style.css         - Styling for all pages
script.js         - Handles login & logic
book.js           - Book and genre data
favicon.ico       - Library icon
README.txt        - Project details

MAIN FEATURES
------------------------------------------
1. View all books (10 genres × 30 books each)
2. Issue books with borrower info, issue date & due date
3. View issued book list
4. View returned book history
5. Search bar to find books or borrowers
6. Logout option for session end
7. Responsive design (works on PC & phone)

------------------------------------------
HOW TO RUN
------------------------------------------
1. Extract the "library-project" folder.
2. Open "index.html" in any modern web browser.
3. Wait for splash screen → redirects to "login.html".
4. Enter valid credentials and CAPTCHA to log in.

------------------------------------------
RUN ON PHONE (SAME WIFI)
------------------------------------------
1. Run local server using Python:
   python -m http.server 8000
2. Note your IP address (e.g., 192.168.1.5)
3. Open in phone browser:
   http://192.168.1.5:8000/login.html
